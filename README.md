# Restaurant Menu

A stylish and responsive restaurant menu website showcasing various dishes and beverages. The website includes features like smooth scrolling, category-based filtering, search functionality, and a popup for viewing item details.

## Live Demo

Check out the live demo of the project: [Restaurant Menu][(https://carlos-hash-010164100.github.io/La_OVeja_Negra/)]

## Features

- Smooth scrolling for navigation links.
- Category-based filtering of menu items.
- Search functionality to filter menu items.
- Popup display for ingredients of each menu item.
- Responsive design for optimal viewing on all devices.
- Scroll to top button for easy navigation.

1. Open the website.
2. Open index.html in your web browser to view the project locally.

## Technologies Used

1. HTML: Markup language for structuring the web pages.
2. CSS: Styling the web pages.
3. JavaScript: Adding interactivity and functionality.
4. Font Awesome: Icons for enhancing the UI.
5. Google Maps Embed: Embedding Google Maps.
